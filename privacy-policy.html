Privacy Policy

QBO Reconciliation Tool · Last updated: February 11, 2026

Summary: This is a private, personal-use application. It does not collect, store, share, or sell any user data to third parties. All data remains on your local machine.

1. Application Overview

This application ("QBO Reconciliation Tool") is a private, personal-use integration that connects to QuickBooks Online ("QBO") via Intuit's API solely for the purpose of reconciling the developer's own financial records. This application is not distributed to, or intended for use by, any third parties.

2. Data Collection

This application accesses the following data from your QuickBooks Online account:

  • Transaction records (date, description, amount, category)
  • Chart of accounts
  • Customer, vendor, and invoice records

This data is accessed in real-time via Intuit's API and is not stored in any external database or cloud service. All data processing occurs locally on the developer's personal machine.

3. Data Usage

Data accessed through the QBO API is used exclusively for:

  • Personal financial reconciliation and bookkeeping
  • Generating local reports and summaries
  • Categorizing transactions for personal tax preparation

4. Data Sharing

This application does not:

  • Share data with any third parties
  • Sell or monetize any financial data
  • Transmit data to external servers or cloud services
  • Use data for advertising or marketing purposes

5. Data Storage & Security

All data retrieved from QuickBooks Online is processed locally and is subject to the security measures of the developer's personal device. OAuth 2.0 tokens used for authentication are stored in a local environment file and are never shared or transmitted.

6. Third-Party Services

This application connects to Intuit's QuickBooks Online API. Intuit's own privacy policy governs how Intuit handles data on their platform. See Intuit's Privacy Statement for details.

7. Data Retention

No financial data is permanently stored by this application. Data is accessed on-demand and exists only in local memory during active use. OAuth tokens are retained locally until manually revoked or expired.

8. Your Rights

As the sole user and developer of this application, you retain full control over all data. You may revoke access at any time by disconnecting the application from your QuickBooks Online account via the Intuit Developer Portal.

9. Contact

For questions about this privacy policy, contact the developer directly as the sole operator of this application.

10. Changes to This Policy

This privacy policy may be updated as needed. Any changes will be reflected by the "Last updated" date above.
